# Customer Churn Prediction Project

## Project Overview

This project analyzes customer churn using the Telco Customer Churn dataset. The objective is to identify factors that influence customer attrition and build machine learning models to predict churn.

## Dataset

* Telco Customer Churn Dataset
* 7,043 customer records
* Customer demographics, services, billing information, and churn status

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Power BI

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Converted TotalCharges to numeric format
* Removed invalid records

### 2. Exploratory Data Analysis (EDA)

* Churn distribution analysis
* Monthly charges analysis
* Customer tenure analysis

### 3. Feature Engineering

* One-hot encoding of categorical variables
* Dataset preparation for machine learning

### 4. Machine Learning Models

* Logistic Regression
* Random Forest Classifier

### 5. Model Performance

Logistic Regression Accuracy:

* 78.5%

Random Forest Accuracy:

* 78.5%

### Top Factors Influencing Churn

* Total Charges
* Monthly Charges
* Tenure
* Internet Service (Fiber Optic)
* Electronic Check Payment Method

## Business Insights

* Month-to-month customers are more likely to churn.
* Fiber optic customers show higher churn rates.
* Customers with higher monthly charges are more likely to leave.
* Electronic check users exhibit higher churn.
